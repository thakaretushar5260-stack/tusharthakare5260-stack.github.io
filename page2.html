 Sign magnitude addition is performed using an Arduino Nano and LEDs. The most significant bit LED (red) indicates the sign, while the other LEDs represent the magnitude. Arduino processes sign magnitude logic, computes the sum, and displays the result visually. This project is ideal for understanding digital arithmetic using sign magnitude representation through a simple hardware and code interface.

Design a 2-bit Up/Down Counter: Implement a counter that can count both up and down, with output display using LEDs.

 Component List :
(2-bit Up/Down Counter with LED Display) Breadboard Digital IC (counter IC such as 74LS193/40193 or microcontroller)
LEDs (2, each represents one output bit)
Resistors (one per LED, ~220Ω)
Push buttons (for UP and DOWN control)
Jumper wires (various colors)
Power supply (9V battery or USB source)
Optional: MOSFET/regulator for stable power​​


Pressing the UP button increments the binary count (00, 01, 10, 11), while the DOWN button decrements it (11, 10, 01, 00). The counter outputs are sent to two LEDs—each LED lights up for a binary '1' and is off for '0', visually showing the count on the breadboard every time a button is pressed. 

Arduino Turbidity Water Quality Monitoring System

Arduino Turbidity Water Quality Monitoring System

Component :
Arduino Uno 
Turbidity Sensor Module
16x2 LCD Display 
Connecting wires
Power supply/battery 
LEDs (optional indicators)

This project uses an Arduino Uno and a turbidity sensor module to measure water turbidity or clarity. The sensor's analog output is read by the Arduino, which processes the signal and displays the turbidity level on a 16x2 LCD. Optional LEDs indicate status, and the setup can be calibrated and tested using the Arduino Serial Monitor. This system is useful for monitoring water quality in various environments.
